Raku in the browser

Three little apps, each powered by rakupp compiled to WebAssembly — the same interpreter as the CLI, running entirely in your browser. No server, no JavaScript port: each app feeds your input to a Raku program and shows what it prints.

📝 Live Markdown

Type Markdown, see it rendered as you go — by the markdown showcase's Raku grammar, not a JS library.

🔧 JSON Formatter

Beautify or minify JSON with the json showcase's parser and serializer (keys sorted for stable output).

🔍 Regex & Grammar

Test a Raku regex with live match highlighting, or watch a full grammar parse into a tree — Raku's own engine.

These reuse the CLI showcase programs directly, and bundle the Raku.js WebAssembly build (rakujs/build.sh, copied in by serve.sh). Servers (pastebin, chat, …) can't run here — the sandbox has no sockets.
